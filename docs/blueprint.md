# **App Name**: Finance Flow

## Core Features:

- Transaction Input: Record income and expenses with categorization, dates, and amount details.
- Scheduled Transactions: Manage single, installments and reoccurring transaction events, scheduling payments.
- Firebase Integration: Use Firebase Authentication and Firestore for user data storage.
- Gamified XP System: Users earn XP by marking bills as paid early. The XP progress determines user's level, such as Beginner, Saver, Finance Master.
- Monthly Summaries: Overview of Total Payable, Total Paid and Remaining Balance on a monthly basis.
- Transaction Classification Tool: Leverage a tool powered by LLMs that intelligently predicts transaction categories based on descriptions.
- Visual Achievements: Celebrate monthly financial milestones with animations and visual rewards for managing finances efficiently.

## Style Guidelines:

- Primary color: Soft teal (#64CCC9) to promote tranquility and growth.
- Background color: Light beige (#F0EAD6), offering a clean, unobtrusive backdrop.
- Accent color: Muted terracotta (#E29578) for highlighting CTAs.
- Font pairing: 'Poppins' (sans-serif) for headers and short texts, 'PT Sans' (sans-serif) for body text, prioritizing readability.
- Consistent use of Lucide-React for its simplicity and clarity.
- Adoption of minimalist design: extensive usage of white space and an arrangement focused on clarity.
- Provide micro-interactions in the interface to respond to user inputs.
